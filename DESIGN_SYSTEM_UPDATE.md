## 色彩系统全面升级

### 深色主题（默认）
- **主色** - 专业靛蓝：`#1E5AA8`（信任、稳定、专业）
- **强调色** - 振动青色：`#00D9FF`（成长、行动、活力）
- **积极色** - 翡翠绿：`#10B981`（上升、收益）
- **消极色** - 深红：`#FF3B3B`（下降、风险）
- **背景色** - 极深蓝黑：`#04080F`（清晰、沉稳）
- **表面色** - 深蓝灰：`#0A0F1D`（高对比度）

### 浅色主题
- 保持一致的蓝色系统，但使用更浅的变体
- 背景：`#F9FAFB`（干净、专业）
- 表面：`#FFFFFF`（纯白、高对比）

## 字体系统升级

### 字体选择
- **显示字体**：Plus Jakarta Sans（现代几何、专业、金融级）
  - 优点：高可读性、现代感、几何线条感
  - 用途：标题、按钮、导航标签

- **代码字体**：Space Mono（技术感、精确）
  - 用途：数据展示、时间戳、代码

### 字体权重优化
- 常规：400（正文）
- 中等：500（辅助内容）
- 半粗：600（强调）
- 粗体：700（标题）
- 超粗：800（LOGO、主标题）

## 技术实现细节

### CSS变量体系
```css
/* 颜色 */
--color-primary: #1E5AA8
--color-accent: #00D9FF
--color-success: #10B981
--color-danger: #FF3B3B

/* 背景和表面 */
--color-background: #04080F
--color-surface: #0A0F1D
--color-surface-elevated: #0F1428

/* 文本 */
--color-text-primary: #F5F7FB
--color-text-secondary: #A0ABB8
--color-text-muted: #6B7C8F

/* 阴影 */
--shadow-xs: 0 1px 2px rgba(0, 0, 0, 0.5)
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.4)
--shadow-md: 0 4px 16px rgba(0, 0, 0, 0.5)
--shadow-lg: 0 12px 32px rgba(0, 0, 0, 0.6)
--shadow-xl: 0 20px 48px rgba(0, 0, 0, 0.7)
```

### 玻璃化效果

## 性能考虑

- 使用CSS变量实现主题切换（无JavaScript成本）
- 优化的阴影效果（多层阴影但计算量有限）
- 平滑的过渡（150-300ms，不影响性能）
- 玻璃化效果使用backdrop-filter（现代浏览器支持）
