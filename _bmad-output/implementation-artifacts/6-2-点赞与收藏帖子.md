# Story 6.2: 点赞与收藏帖子

Status: ready-for-dev

<!-- Note: Validation is optional. Run validate-create-story for quality check before dev-story. -->

## Story

作为已登录用户，
我希望点赞和收藏感兴趣的帖子，
以便表达认可并方便后续回看。

## Acceptance Criteria

1. **Given** 用户点击帖子的点赞按钮
   **When** 调用 `POST /api/v1/posts/:id/like`
   **Then** `post_interactions` 表插入记录（`type='like'`），`posts.likes_count +1`（FR36）
   **And** UNIQUE(user_id, post_id, type) 约束保证同一用户不重复点赞
   **And** 返回格式 `{"liked": true, "likes_count": N}`

2. **Given** 用户再次点击已点赞帖子的点赞按钮（取消点赞）
   **When** 再次调用 `POST /api/v1/posts/:id/like`
   **Then** `post_interactions` 中对应记录删除，`posts.likes_count -1`
   **And** 返回格式 `{"liked": false, "likes_count": N}`

3. **Given** 用户点击收藏按钮
   **When** 调用 `POST /api/v1/posts/:id/collect`
   **Then** `post_interactions` 表插入记录（`type='collect'`），支持取消收藏（FR38）
   **And** 返回格式 `{"collected": true}`

4. **Given** 用户访问"我的收藏"列表
   **When** 调用 `GET /api/v1/users/me/collections`
   **Then** 返回当前登录用户收藏的帖子列表，支持分页（page/per_page）
   **And** 每条帖子含完整数据（content、author、strategy card 等）

5. **Given** 用户请求的 post_id 不存在
   **When** 调用任意互动端点
   **Then** 返回 `404` 错误

6. **Given** 未认证用户
   **When** 调用任意互动端点
   **Then** 返回 `401` 错误

### 并发处理：原子性更新

`likes_count` 更新需要注意并发安全：

```python
# ⚠️ 不推荐（有竞态风险）
post.likes_count += 1

# ✅ 推荐（数据库原子更新）
from sqlalchemy import text
db.session.execute(
    text("UPDATE posts SET likes_count = likes_count + 1 WHERE id = :id"),
    {"id": post_id}
)
# 或使用 SQLAlchemy 表达式
Post.query.filter_by(id=post_id).update(
    {Post.likes_count: Post.likes_count + 1}
)
```

### 乐观更新前端模式

点赞/收藏使用乐观更新，避免界面延迟：
```typescript
// useCommunityStore.ts
async toggleLike(postId: string) {
  const post = this.posts.find(p => p.id === postId)
  if (!post) return

  // 乐观更新
  const wasLiked = this.likedPostIds.has(postId)
  if (wasLiked) {
    this.likedPostIds.delete(postId)
    post.likes_count--
  } else {
    this.likedPostIds.add(postId)
    post.likes_count++
  }

  try {
    const result = await likePost(postId)
    // 用服务端返回的准确值替换
    post.likes_count = result.likes_count
    if (!result.liked) this.likedPostIds.delete(postId)
    else this.likedPostIds.add(postId)
  } catch (e) {
    // 回滚
    if (wasLiked) {
      this.likedPostIds.add(postId)
      post.likes_count++
    } else {
      this.likedPostIds.delete(postId)
      post.likes_count--
    }
  }
}
```
